# 🚗 DriveLux Rentals – Premium Car Rental Frontend

A modern, fully responsive premium car rental website built with **HTML5**, **CSS3**, and **Vanilla JavaScript (ES6)**. No frameworks, no backend, no dependencies – just clean, modular code perfect for a college placement portfolio or a frontend developer showcase.

![DriveLux Screenshot](https://images.unsplash.com/photo-1503376780353-7e6692767b70?w=1200&q=80)

---

## ✨ Features

- **11 Dynamic Pages** (SPA via hash routing): Home, Cars, Car Details, Booking, Wishlist, My Bookings, Login, Register, About, Contact, 404
- **Advanced Car Search & Filtering**  
  Filter by brand, fuel, transmission, seats, price; sort by price or rating – all rendered dynamically from a local JSON array.
- **Complete Booking Flow**  
  Auto‑calculates rental days, base price, extras (insurance, GPS, child seat), tax, discounts, and grand total.
- **Wishlist & Booking Management**  
  Save favorite cars and manage bookings using **Local Storage** (no server needed).
- **Dark / Light Mode Toggle**  
  Preference saved automatically.
- **Interactive UI Elements**  
  Image sliders, FAQ accordion, animated counters, toast notifications, scroll‑to‑top button, and smooth page transitions.
- **Glassmorphism & Neumorphism**  
  Premium visual style with soft shadows, rounded corners, and subtle animations.
- **Fully Responsive**  
  Looks great on desktop, tablet, and mobile.
- **Accessibility & SEO Friendly**  
  Semantic HTML5, aria‑labels, meta tags, and smooth scroll.

---

## 🛠️ Technologies Used

- **HTML5** – Semantic, accessible markup
- **CSS3** – Custom properties, Flexbox, Grid, animations, glassmorphism
- **JavaScript (ES6 Modules)** – DOM manipulation, event handling, hash‑based routing, Local Storage, template literals, array methods (map, filter, sort)
- **Google Fonts** (Inter & Playfair Display)
- **Font Awesome 6** (icons)

Absolutely **no** React, Angular, Vue, Bootstrap, Tailwind, jQuery, or any backend.
